
    Java 8 (функционалльность и декларативность(т.е. описание необходимого результата сразу))

    Java 8 привнесла в язык функциональный подход, реализованный по типу как
    это реализовано в функциональных языках, например JS.
    Фукнциональный подкход заключается в том что наконец-то, несомтря на парадигму ООП,
    в java стало возможным рассматривать функции в отрыве от классов.
    Стало вомзможным передавать ссылки на функции и использовать колбэки.
    Ссылка на функцию содержит адрес не блока памяти со значением, а ссылку на сам код функции.
